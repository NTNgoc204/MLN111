# Huong Dan Chay Bai Tap (Cho Nguoi Khong Biet Lap Trinh)

Ban co 2 cach de xem bai:

## Cach 1 (de nhat): Mo link online

Neu nhom da deploy len Vercel, chi can mo link web, khong can cai dat gi.

## Cach 2: Chay truc tiep trong folder nay (Windows)

### Buoc 1: Cai Node.js (chi lam 1 lan)

1. Mo trang: https://nodejs.org
2. Tai ban **LTS** va cai dat mac dinh.

### Buoc 2: Chay website

1. Mo folder nay.
2. Bam dup file **CHAY_WEBSITE.bat**.
3. Cho 1-2 phut lan dau (vi can cai thu vien).
4. Trinh duyet se mo tai dia chi: http://localhost:5173

Neu khong tu mo trinh duyet, copy dia chi `http://localhost:5173` va dan vao Chrome/Edge.

## Neu gap loi

- Loi bao khong tim thay `node`/`npm`: chua cai Node.js hoac can khoi dong lai may sau khi cai.
- Cong `5173` da bi dung: dong cua so cu, roi chay lai file `.bat`.

## Cach tat website

Quay lai cua so den (Command Prompt) va bam **Ctrl + C**.
